# Gradle Demo Project with Jackson Core

This is a demo Gradle project using Jackson Core version 2.17.3.

## Building the Project

```bash
./gradlew build
```

## Running the Demo

```bash
./gradlew run
```

Or run the main class directly:

```bash
./gradlew run --main-class=com.example.Demo
```

## Dependencies

- Jackson Core: 2.17.3
- JUnit 5 (for testing)
